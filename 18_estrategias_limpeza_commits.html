Boas práticas 1/3 · Comunicação clara nos commits

Estratégias de limpeza de commits

> Rebase interativo (git rebase -i): reescrever o histórico local — juntar (squash/fixup), reordenar ou editar mensagens antes de abrir o PR.
> git commit --amend: ajustar o último commit (mensagem ou arquivos esquecidos) sem criar um commit “corrige o commit anterior”.
# Rebase interativo (ex.: últimos 3 commits)
git rebase -i HEAD~3

# Corrigir mensagem ou incluir mudanças no último commit
git add .
git commit --amend --no-edit

# Só alterar a mensagem do último commit
git commit --amend
