Ferramentas que facilitam a jornada

Commitizen
Interface interativa para criar commits padronizados sem precisar decorar a especificação.
Husky + Commitlint
Bloqueie commits que não seguem o padrão localmente, antes mesmo de chegarem ao servidor.
Semantic Release
Automatize totalmente o versionamento e a geração de notas de versão baseada nos commits.
# Instalando as ferramentas
npm install -D commitizen
npm install -D @commitlint/cli

# Configurando o Husky
npx husky add .husky/commit-msg

# Exemplo de uso
git cz
? Select the type of change:
> feat: A new feature
fix: A bug fix
