Boas práticas 1/3 · Comunicação clara nos commits

git commit --amend

> O --amend altera somente o último commit: dá para corrigir a mensagem ou incluir arquivos que faltaram, sem empilhar um commit do tipo “corrige o anterior”.
# Incluir mudanças no último commit (mantém a mensagem)
git add .
git commit --amend --no-edit

# Só alterar a mensagem do último commit
git commit --amend
