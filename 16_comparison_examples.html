Exemplos: Do Caos ao Controle

O Caos
$ git commit -m "..."
"ajustes no login"
$ git commit -m "..."
"nova feature"
$ git commit -m "..."
"fix bug"
O Controle
$ git commit -m "..."
fix(auth): expiração de token no Safari
$ git commit -m "..."
feat(api): endpoint de busca por categoria
$ git commit -m "..."
test(ui): validar responsividade do menu
